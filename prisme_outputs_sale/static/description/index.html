Description and Features

Add specific reports (for paper or e-mail) and features needed to these reports related to sales.

Module Features :

  • Sale order - Form View
    • Print Totals checkbox : to show or not the totals on the report
    • Print VAT checkbox : to show or not the VAT on the report
    • Header comment field : to add comment in report header
    • Footer comment field : to add comment in report footer
  • Sale order Report for Paper / E-Mail
    • Don't show discount if equals 0
    • Without Logo and footer for Paper
    • Misc. graphical changes







Installation

Prerequisites :
  • Nothing
Steps :
  1. Copy the content of "addons_folder"/prisme_outputs_sale/TODO_manually/header_footer.rml into the "Report Configuration" Tab of your Company (Menu / Settings / Companies / Companies)
  2. Click the install button and it's done

Configuration

To configure report logo
  1. go to Setting/Compagnies/Companies
  2. select your compagny
  3. edit and change logo
To configure report footer
  1. Go to the General Information of your Company (Menu / Settings / Companies / Companies)
  2. Custom Footer
  3. Edit and add your compagny informations (for example : "Rte de Montagny 11 - 1772 Ponthaux - Tel. 026 475 26 70 - E-mail: info@prisme.ch - www.prisme.ch") in Report Footer
