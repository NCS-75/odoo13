Description and Features

This module reimplements timesheet invoicing as it was in Odoo 8.

Module Features :

  • Timesheet activities - form view
    • Link to the invoice
    • Type of invoicing
  • Timesheet invoices - form view
    • Analytic account
  • Timesheet invoicing
    • Invoicing of timesheets like Odoo 8.0 (From project/invoicing), fills the 'project_id' field to make a link back to the project
