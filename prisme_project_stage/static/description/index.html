Description and Features

Add functionalities to improve the contract view of the Fleet module with some kilometers stats

Module Features :

  • Vehicles tab - Vehicles Contract - Form view
    • Add new last entry field : last odometer entry by its date
    • Add new contract duration computed field : (contract expiration - contract start)
    • Add new planned Kilometers per year text field : number of kilometers per year according to the contract
    • Add new planned Kilometers per month computed field : (number of kilometers per year / 12)
    • Add new planned total Kilometers computed field : (number of kilometers on the contract * contract duration)
    • Add new total Kilometers performed computed field : total number of kilometers performed
    • Add new current Kilometers performed this year computed computed field : number of kilometers performed this year according to the contract with a progress bar
    • Add new current Kilometers performed this month computed field : number of kilometers performed this month according to the contract with a progress bar
