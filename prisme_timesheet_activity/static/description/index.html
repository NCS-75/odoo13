Description and Features

This module re-implements timesheets as they where in Odoo 10.
A new field to set expected hours for the period is created, allowing the computation of a bonus/malus depending on the effective hours
It also adds a full form and list view to edit the timesheet activities easily.
New fields are created to allow employees to set the start and end times of each activities and to describe the activities.

Module Features :

  • Re-implementation of Odoo 10 style timesheets
  • Timesheet activities - form view
    • Start and end time with interval time calculation
    • Internal description field
    • Hours per period
    • Automatically change the partner when the project is modified
  • Timesheet activities - list view
    • Customer
    • Start time (Beginning)
    • End time (End)
    • Filter : This Month
    • Group by : Customer
