Description and Features

Add functionalities to significantly improve the management and the use of timesheets such as start and end time with interval time calculation and several reports.

Module Features :

  • Timesheet activities - form view
    • Start and end time with interval time calculation
    • Internal description field
    • Hours per period
    • Automatically change the partner when the project is modified
    • Prevents the user from saving if a project with tasks is selected and no task is selected
  • Timesheet activities - list view
    • Customer
    • Start time (Beginning)
    • End time (End)
    • Filter : This Month
    • Group by : Customer
  • Reports
    • Project - Time checklist
    • User - Time checklist
    • Invoice - Services description
  • Timesheet invoicing
    • Invoicing of timesheets like Odoo 8.0 (From project/invoicing), fills the 'project_id' field to make a link back to the project
