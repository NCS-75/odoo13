Description and Features

Add functionalities to significantly improve the management of Sale Orders

Module Features :

  • Sale Order / Quotations - Form View
    • Reference/Description: used to reference the client
    • Rounding on Subtotal field : to choose the subtotal rounding (for example 0.05)
    • Creation Date field : read only - quotation creation date
    • Confirmation Date field : read only - quotation confirmation date
    • Quotation Validity datefield
    • Quotation Comment field
    • Notes field : to write something about the Sale Order
    • Cancellation Reason field : required if the SO/quotation is canceled
    • Confirmation date field : do not copy when duplicating Sale Order
    • Order line form
      • Delay field : alternative to Delivery Date (automatically calculate from Delivery Date)
      • Refused field : to save a partial refused quotation
      • Refusal Reason field
      • Status : display related sale order status
      • Discount amount field : add possibility to apply a discount in amount
  • Sales Analysis
    • new columns
      • Sale Order
      • Total Discount
      • Total Price (Net)
      • Total Purchase
      • Total Margin
    • Search by sale order

Frequently asked questions (FAQ)





How to use discounts ?

Discounts are not enabled by default, you can enable it in Settings/Configuration/Sales -> Product Features -> 'Allow setting a discount on the sales order lines'
