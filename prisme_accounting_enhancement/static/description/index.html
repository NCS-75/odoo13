Description and Features

Add functionalities to significantly improve the management of financial/analytic accounting

Module Features :

  • Product
    • Possibility to set an analytic account (via dependent module "account_analytic_default")
  • Account invoice
    • Possibility to select a discount in percent and/or in amount for each line
    • Possiblity to round subtotals
