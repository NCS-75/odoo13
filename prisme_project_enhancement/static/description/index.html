Description and Features

This module adds an 'amount', a 'balance_previous_command' and an 'account' field to the Tasks of the Projects application.

Module Features :

  • Adds an 'amount', a 'balance_previous_command' and an 'account' field to the Tasks of the Projects application
  • They respectively are used to store : the allocated amount for the task, the balance from a previous command for the task and to link a task to an invoice line
  • The 'analytic_account' field that was added to invoices in v10 is now added by the prisme_invoicing_timesheet module
