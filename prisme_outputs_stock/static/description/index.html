Description and Features

Add specific reports (for paper or e-mail) and features needed to these reports related to stock picking.

Module Features :

  • Stock picking view : add new Transporter field
  • Picking List for Paper / E-Mail
    • Transporter
    • Shipped quantity
    • Back order quantity
    • Delivery Method
    • Shipping Date
    • References
    • Without Logo and footer for Paper
    • Misc. graphical changes
